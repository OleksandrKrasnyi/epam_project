## Installation

Software prerequisites:

- [Python](https://python.org)
- [Pip Package Manager](https://pypi.python.org/pypi)

### Running the app

To run the application, in it's folder
- install venv
- activate venv
- install all [requirements](requirements.txt) from requirements.txt
- in a console type:
```bash
python app.py
```
